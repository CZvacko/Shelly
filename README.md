# Here are some scripts for Shelly  

Shelly Pro 3 EM - [Watchdog - exported power](/Scripts/Watchdog%20-%20Exported%20power.js)  
It can monitor exported power (to the grid) and notify user via Shelly Cloud Scene. It can also control switch add-on for controlling some other devices (relay/inverter). With a minor modification, it can also monitor whether imported energy exceeds a certain value.
